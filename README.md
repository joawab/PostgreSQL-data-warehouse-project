# PostgreSQL-data-warehouse-project

PostgreSQL data warehouse implementing the bronze/silver/gold medallion architecture - from raw CSV ingestion to analysis-ready data.



Objective



Develop a modern data warehouse using SQL Server to consolidate sales data, enabling analytical reporting and informed decision-making.

Specifications



&#x20;   Data Sources: Import data from two source systems (ERP and CRM) provided as CSV files.

&#x20;   Data Quality: Cleanse and resolve data quality issues prior to analysis.

&#x20;   Integration: Combine both sources into a single, user-friendly data model designed for analytical queries.

&#x20;   Scope: Focus on the latest dataset only; historization of data is not required.

&#x20;   Documentation: Provide clear documentation of the data model to support both business stakeholders and analytics teams.
